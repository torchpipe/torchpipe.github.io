# Ppl.cv相关后端

`version>=0.3.3b2`

需要编译时启动[相关选项](../installation.mdx#compilation-options)

```bash
BUILD_PPLCV=1 pip install -e .
```

|[PPLResizeTensor]|resize_h,resize_w | resize_h[int],resize_w[int]|data[Tensor]|result[Tensor]||
